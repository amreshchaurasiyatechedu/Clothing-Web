# 🛍️ Clothing Store

A modern e-commerce website for a clothing store built with HTML, CSS, and JavaScript.

## 📋 Features

- **User Authentication** - Login/Signup system
- **Product Catalog** - Browse clothing items with images and prices
- **Shopping Cart** - Add/remove items, quantity management
- **User Profile** - View order history and manage account
- **Admin Dashboard** - Manage products and view orders
- **Order Receipts** - Generate and view order receipts

## 🏗️ Project Structure

```
clothing-store/
├── assets/
│   ├── css/          # Stylesheets
│   ├── js/           # JavaScript files
│   └── images/       # Product and static images
├── pages/            # HTML pages
│   ├── shop.html     # Product listing page
│   ├── cart.html     # Shopping cart
│   ├── profile.html  # User profile
│   ├── receipt.html  # Order receipt
│   └── admin.html    # Admin dashboard
├── index.html        # Login/Signup page
├── style.css         # Main stylesheet
└── script.js         # Main JavaScript
```

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/clothing-store.git
   ```

2. **Open the project:**
   - Navigate to the project folder
   - Open `index.html` in your browser

3. **Start shopping:**
   - Login or sign up
   - Browse products in the shop
   - Add items to cart
   - Complete checkout

## 💻 Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Flexbox/Grid
- **JavaScript** - Client-side functionality
- **LocalStorage** - Data persistence
